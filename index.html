What is this?

This script shall help to keep various CVS-files (comma separated value) in sync / spott the differences. These CVS-files may originate from various systems.

How to use this

Files have to following specs:

  • Filename is <date>_<systemname>[_<verions>].cvs. <date> has to be in the format YYYYDDMM.
  • Files may contain empty lines or comments. Comments begin with "#".
  • One item per line. The first Element of each line will function as syncronization key between all provided files.
  • Elements within one line have to be seperated by either "," or ";"

As soon as you provide files, their names will appear here
